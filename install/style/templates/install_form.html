OpenBooking Installation

Welcome to OpenBooking. This page is designed to automate installation and configure the 'config.php' file, and will perform database initialisation. This file can be edited manually after installation, and contains database and administration information for the application.

Please be aware that your PHP installation must have the php-mysql and php-ldap extensions installed. To install OpenBooking, you must have the following information:

  • Database (MySQL)

  • Hostname - the address of the server
  • Database - the name of the database on the server
  • Table Prefix - the prefix that you would like to use for OpenBooking database tables. Defaults to "ob_".
  • Username and Password - the login credentials OpenBooking will use to connect to your database

  • LDAP

  • Hostname - the address of the LDAP server to use for authentication
  • Base DN - the base lookup within your LDAP directory that will contain your users
  • Administrator username(s) - the usernames of network users that you would like to add as administrators

You can test the log-in credentials for both the Database and LDAP servers by clicking the appropriate button on the right. When you have verified both sets of credentials, click "Install" to install OpenBooking.
